# ModLoader64-GUI-Imgui
Remake of the GUI in ImGui


# Running:
Install Node
Run 
`npm install -g yarn typescript gulp-cli`
`npm install`
`npm run build`

You also need libgtk to run / test it
Ubuntu/Debian:
```
sudo apt-get update
sudo apt-get install libgtk-4-1
```
Fedora:
```
sudo dnf install gtk4
```
Arch Linux/Manjaro:
```
sudo pacman -Syu gtk4
```
to test run
`npm test`
